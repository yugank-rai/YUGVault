### ğŸš€ About The Project

**AlgoVault** is a sleek, dark-themed web application that acts as a personal database for your coding problems. Instead of using messy spreadsheets, it allows you to save problem links, attach key "tricks" or notes, and filter through your collection instantly.

### âœ¨ Key Features

* **Live Search:** Instantly filter your collection by Problem Name or Tag (e.g., "DP", "Arrays") without reloading.
* **Data Persistence:** Uses **LocalStorage** so your data is saved automatically and stays there even if you refresh or close the browser.
* **Smart Tagging:** Categorize problems with tags like Arrays, DP, Trees, and Graphs.
* **Trick Tracker:** A dedicated section to save the specific "Aha!" moment or trick for every problem.
* **Responsive Design:** Optimized for desktop and mobile, featuring a modern Glassmorphism UI.
